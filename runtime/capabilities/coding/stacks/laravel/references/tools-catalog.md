# Laravel Tool Guides

These guides are copied into onboarded projects under `docs/tools/laravel/`.

## Included Guides

- `artisan.md`: preferred scaffolding and command discovery
- `boost.md`: Laravel Boost MCP usage and debugging order
- `code-quality.md`: formatting, lint, and validation workflow
- `playwright.md`: safe browser validation guidance
- `sail.md`: container-first execution patterns when Sail exists
- `security-review-checklist.md`: stack-aware pentest checks for auth, runtime, and secrets
- `testing.md`: test command selection and release checks

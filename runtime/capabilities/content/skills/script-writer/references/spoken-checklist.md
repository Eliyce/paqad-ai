# Spoken Script Checklist

1. Read every VO aloud once before saving.
2. No syllable-heavy mouthfuls — replace with short Anglo-Saxon equivalents.
3. No bare "this" / "that" — name the antecedent.
4. Transitions are audible bridges, not silence.
5. Numbers in words ("twelve") for speakable copy; digits ("12") only when the visual carries them.
6. Avoid sentences over ~20 words — listeners run out of breath.
7. Time the segment boundaries; total runtime within ±10% of the brief target.

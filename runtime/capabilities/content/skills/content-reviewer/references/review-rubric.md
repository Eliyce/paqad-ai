# Review Rubric

Walk every heuristic; never skip silently.

1. **Clarity** — does each sentence state one idea the audience can act on?
2. **Structure** — does each section deliver on its heading promise?
3. **Evidence** — are claims supported with specifics (numbers, examples, links)?
4. **Audience fit** — is jargon level matched to the reader role?
5. **Voice** — active over passive; concrete over abstract.
6. **Mechanics** — links resolve, no broken references, no `click here`.
7. **Style compliance** — matches `docs/instructions/rules/writing-style.md` when present.
8. **Accessibility** — alt text for images, link text describes destination, headings nest sensibly.
